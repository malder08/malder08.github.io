Google Code-In

  • Time Frame: December 2019 - January 2020
  • Worked with open-source companies by completing tasks for them
  • Chose to complete tasks for TensorFlow
  • Completed a total of nine tasks for TensorFlow, creating neural networks, writing tutorials, and contributing to documentation

Task 1: [Advanced] Add a usage example to TensorFlow 2.0 API documentation. Added a usage example for the log_sigmoid operation within TensorFlow.

Task 2: [Advanced] Upgrade a TensorFlow 1.x tutorial using tf.keras to TF 2.0. Updated a tutorial that used TensorFlow 1.0 in creating a convolutional neural network with Keras to use TensorFlow 2.0. The original tutorial can be found here: Machine Learning Mastery Tutorial. The updated tutorial created by me was created in a Google Colab and can be found here: Google Colab.

Task 3: [Advanced] Upgrade a StackOverflow answer using tf.keras to TF 2.0. Updated a StackOverflow that had previously used stand-along keras and Tensorflow 1.0 to TensorFlow 2.0 and keras. The question concerned an issue with the input of neural network. The question and updated answer can be found here: StackOverflow Q&A.

Task 4: [Advanced] Solve a machine learning problem using a UCI data set.. Used a University of California Irvine dataset in order to solve a machine learning problem. The neural network I created was capable of classifying how many solar flares would occur in a 24 hour period of three different classes. The predictions were based on ten different pieces of data. The Google Colab can be found here: StackOverflow Q&A.

Task 5: Build a simple Auto-encoder model using tf.keras. Created an auto-encoder which is able to condense images and then reconstruct them based off of that condensed form. The model was trained on the CIFAR-10 dataset. The Google Colab with th auto-encoder can be found here: Google Colab.

Task 6: [Advanced] Train a model using both tf.keras and tf.estimator.. Created a neural network using both tf.keras and tf.estimator in order to compare the different methods. The neural network was created with the Sloan Digital Sky Survey dataset and was able to determine whether an obesrvation of space was a star, galaxy, or quasar with 80% accuracy based off of 15 different photometric and spectral data. The Google Colab can be found here: Google Colab.

Task 7: Complete the Udacity TensorFlow course track.. Completed the TensorFlow's Udacity course which taught me the basics of machine learning and neural networks.

Task 8: Tutorial for Polynomial Regression. Created a tutorial on Polynomial Regression that showed people the math behind Polynomial Regression, how to generate a random dataset with python, how to plot that data, how to do Polynomial Regression with that dataset, and how to avoid over-fitting and under-fitting. The Google Colab with the tutorial can be found here: Polynomial Regression Tutorial.

Task 9: Create and open-source an Android application using TensorFlow Lite. Created a mobile application with Flutter that could classify the contents of images. The model I used was SSD MobileNet. The Github repository with the open-source app can be found here: Github Repository.
